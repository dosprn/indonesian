Uraian modul Bahasa

Para pemakai bisa memodifikasi modul yang sudah ada atau menciptakan milik mereka sendiri. Modul bahasa adalah satu file teks dengan extention .LNG bahwa meliputi string dari jenis di bawah:

short_sentence_definition=sentence_translation

Dalam contoh ini, "definisi kalimat" adalah satu nama sudah dikenal dari kontrol windows DOSPRN dan anda tidak bisa memodifikasi string ini. "Translasi kalimat" adalah satu translasi dari nama kontrol DOSPRN ke dalam bahasa anda. Anda dapat dengan bebas mengubah string ini. 

Anda bisa mengcopy berkas (file) modul bahasa apapun ke satu berkas (file) baru (tetapi dengan extention sama) dan membuka itu dengan Notepad (atau satu editor teks berbeda). Kemudian anda bisa memodifikasi apapun "translasi kalimat" dan merekam (save) file.

Setelah anda telah membaharui berkas (file) bahasa anda, anda harus restart DOSPRN dan modul bahasa anda akan terisi.

Jika anda menciptakan satu modul interface baru atau memodifikasi yang sudah ada, mempersilakan mengirimkan itu sampai kami. Kami akan gembira untuk mencakup itu didalam versi berikutnya dari DOSPRN berkenaan dengan kebebasan sedemikian rupa sehingga para pemakai lain bisa menggunakan modul bahasa ini.
