Tentang DOSPRN

Program ini didesign untuk mendukung satu pencetakan program DOS ke ink jet dan laser printer modern. Banyak printer murah tidak dapat mencetak dalam mode teks, tetapi hanya mencetak dalam mode grafik (jadi disebut GDS device). Serta, program dos lama ( dan beberapa lebih baru diciptakan pada basis dari mode teks printer matrix. Selanjutnya, banyak printer tidak dapat mensupport bahasa asli dalam mode teks. .

DOSPRN membantu untuk mengakhiri masalah ini. Anda dapat menggunakan ini dengan program DOS dan mengkombinasikan printer.

DOSPRN didistribusikan sebagai Shareware Anda dapat pelajari bagaimana menggunakan DOSPRN selama waktu percobaan. Setelah periode ini, anda harus registrasi copyan anda atau hapus ini secara permanen dari komputer anda dengan utility uninstall pada Group Program DOSPRN. .

Installasi

Download dan jalankan file extract DOSprn190.exe . Anda akan ditanyai beberapa pertanya-pertanyaan sederhana. Selama instalasi, DOSPRN mengcopy semua file yang diperlukan dan shortcut ke komputer anda. .

Program Start

Anda dapat memulai DOSPRN dengan mengklik shotcut pada desktop anda atau pada forlder "Statup" (dalam hal ini anda telah memilih opsi ini selama penginstallan). Setalah menjalan DOSPRN, anda dapat menemukan iconnya pada area tray (dekat jam system, biasanya pada pojok kanan bawah), DOSPRN siap untuk bekerja. .

Fitur-fitur Program

Jika anda klik kanan icon DOSPRN pada tray menu, anda akan melihat satu menu windows kecil. Menu ini memberikan akses ke opsi program utama. Opsi-opsi menu :

  • Buka - membuka windows propertis  
  • Fiel Cetak mencetak satu file
  • Print job - mencetak milik anda (jika "mencetak automatis" pilihan ini tidak berjalan)
  • Orientasi - mengubah orientasi kertas
  • Membersihkan cetakan - menghapus yang sedang dicetak pada antrian DOSPRN (jika 'otomatis mencetak' opsi ini tidak berjalan)
  • Batal - menutup menu popup
  • Keluar - menutup DOSPRN

 

Jendela utama properti

Job Time-Out

 Hal ini adalah sejumlah waktu ketidakaktifan sebelum mulai pencetakan. Jika aplikasi-DOS tidak mengirimkan apapun ke printer selama perioda ini, DOSprn mulai pencetakan. 

Otomatis Mencetak

 Jika pilihan ini dimungkinkan, DOSPRN secara otomatis mulai mencetak pekerjaan anda sekarang setelah periode time out Jika tidak, anda bisa menyiapkan pekerjaan pencetakan anda secara manual. .

Cetakan di atas/terhadap Form Feed

 Jika applikasi DOS anda memberikan simbol FORMFeed setelah tiap-tiap tugas pencetakan, anda dapat mencek pilihan ini dan menimizekanwaktu untuk memulai pencetakan Sebagai satu hasil, anda dapat mempercepat prosesnya.

Capture Port

 DOSPRN menankap (mengkapture) semuada data yang dikontrol dari applikasi DOS ker printer dengan penerjemahan penuh data cetakan.

File Cetakan

 Anda bisa hanya mengirimkan satu berkas (file) teks di/dalam ASCII ke printer anda.

Print Job

 Mencetak pekerjaan anda saat ini (jika opsi 'mencetak otomatis' pilihan adalah tidak dipilih)

Bersihkan Pekerjaan

 Menghapus kegiatan anda dari daftar pekerjaan (jika opsi 'otomatis mencetak' tidak dipilih

Kumparan nama file (filemask)

 Nama [dari] berkas (file) (atau topeng [dari] berkas (file)) [yang] berisi data 'buffer' kumparan sementara. 

Catatan: Jika Anda ingin mengubah lokasi nama file kerja, Anda harus terlebih dahulu menonaktifkan opsi Port Interception

Interface Bahasa :

 Pilih bahasa [dari] interface program. Sekarang tersedia interface dalam bahasa Inggris, Perancis,  Deutsch, Bahasa swedia, Belanda, Italiano, Slovak, Espanol, Bangsa Portugis, Yunani, Orang Rumania, Hungarian, Catalan, Croatian, Russia, Ukrainian, Belorussian, Latvian, Cekoslovakia, Bulgaria, Turki dan Polish, Indonesia. Anda dapat menterjemahkan dalam bahasa anda. Silakan baca lagi tentang modul interface bahasa atau kirim email ke supporting.

Orientasi

 Pilih orientasi default kertas

Bungkus baris panjang

 Ketika opsi ini dipilih, DOSPRN akan membungkus semua garis  yang melebihi batas yang telah ditentukan

Menggunakan simbol Form Feed

 DOSPRN bisa yang manapun, baik memproses simbol FormFeed (ASCII 12) atau mengabaikan itu.

LF sebagai CR/LF

 DOSPRN bisa memproses simbol LineFeed (ASCII 10) sebagai sebuah pengembalian sederhana untuk arus pencetakan posisi ke/pada posisi sama di/dalam garis berikutnya, atau sebagai memasangkan CarriageReturn  (ASCII 13) dan LineFeed (ASCII 10). Dalam hal ini posisi pencetakan akan berpindah ke awal dari baris berikutnya.

Dijalankan Minimized

 Jika anda memilih sesi ini, DOSprn akan mulai terminimize pada tray sistem secara default. Selain itu, windows properti utama akan dibuka setelah memulai.

Printer

 Pilih printer saat ini anda. Catatan: jika anda menggunakan printer jaringan anda tidak perlu menunjukan port lokal

SEQUENCES

 Program menonjolkan ESC-sequences pendukungan DOSPRN bisa menandingi berbagai skema mencakup yang sering bekas Epson Esc-P2 men-standar-kan, skema DOSPRN default, HP PCL atau satu skema dapat didefinisikan (user-defined).

Properti huruf-huruf

Huruf-huruf

 Anda dapat meilihat daftar font yang terinstall Huruf-huruf monospace adalah ditampilkan dalam hijau, sementara huruf-huruf proporsional dalam merah. Kami menyarankan bahwa anda menggunakan huruf-hurf monospace (seperti Courier atau Lucida). . 

Hanya Tampilkan Monospace

 Jika Opsi ini dipilih, hanya daftar huruf-huruf akan termasuk huruf monospace

Halaman kode emulasi

Codepage. DOSPRN mendukung codepages 850 (Eropa Timur), 852 (Eropa Barat), 737 (Yunani), 437 (AS), 857 (Turki), 862 (Ibrani), 775 (Latvian), 771 (Lithuanian), 855 (Bulgaria), 1125 (Ukrainian) dan 864 (Arabic). 

"Preview" tombol memungkinkan satu untuk melihat berkas (file) ASCII.tbl dalam codepage ini. Gambar preview harus cocok dengan yang ditampilkan dari file ini dalam kotak DOS. Jika anda biasa dengan ini semua, anda dapat membaca lagi tentang modul-modul stuktur codepage dan memodif sendiri file codepage. Selain itu anda dapat mengirimkan email ke suppoting jika anda memiliki masalah atau kebutuhan khusus. .

Ukuran

 Ukuran default font.

CPI

 Ini adalah parameter untuk mespesifikasikan jumlah karakter per inch. Tidak ada ketergantungan pada gaya dan ukuran huruf.

LPI

 Ini adalah spesifikasi parameter dari jumlah baris per inch. Tidak ada ketergantungan pada gaya dan ukuran huruf.

Margin

 Ukuran margin atas, kiri, bawah dan kanan akan diterapkan ketika pencetakan satu doument teks dari applikasi DOS Tidak terdapat saling ketergantungan di terhadap settingan-settingan dokumen DOS.

Preview

 Sesi preview menampilkan font yang dipilih dalan model WYSIWYG
